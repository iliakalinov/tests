Case 1 (Проверка на заполнение полей)
1. Перейти на сайт tickets.by/gd
2. Ввести место отправления "Минск" 
3. Оставить место прибытия пустым 
4. Нажать кнопку "Поиск"

Ожидаемый результат: Выдача сообщения об ошибки  

Case 2 (Проверка на несуществующие маршруты)
1. Перейти на сайт tickets.by/gd
2. Указать место отправления – «Минкс»
3. Указать место прибытия – «Минск»
4. Указать любую дату 
5. Нажать кнопку "Поиск"

Ожидаемый результат: Неверные данные 


Case 3 (Проверка на работоспособность )
1. Перейти на сайт tickets.by/gd
2. Указать место отправления – «Минск»
3. Указать место прибытия – «Борисов»
4. Указать любую дату 
5. Нажать кнопку "Поиск"

Ожидаемый результат: Вывод списка о возможных поездках


Case 4 (Проверка успешной регистрации)
1. Перейти на сайт tickets.by/gd
2. Нажать на надпись Мой билет 
3. Нажать на кнопку зарегистрироваться 
4. Указать правильные данные для ввода 
5. Нажать кнопку регистрация 

Ожидаемый результат: Успешная регистрация

Case 5 (Проверка на авторизацию)
1. Перейти на сайт tickets.by/gd
2. Нажать на надпись Мой билет 
3. Указать правильные данные для ввода(использованные ранее) 
4. Нажать кнопку вход на сайт

Ожидаемый результат: Успешная авторизация 


Case 5 (Проверка на авторизацию по неверным данным )
1. Перейти на сайт tickets.by/gd
2. Нажать на надпись Мой билет 
3. Указать не правильные данные для ввода
4. Нажать кнопку вход на сайт
Ожидаемый результат: Сообщения вам не удалось авторизоваться 

Case 7 (Проверка валидации при регистрации )
1. Перейти на сайт tickets.by/gd
2. Нажать на надпись Мой билет 
3. Нажать на кнопку зарегистрироваться 
4. Не указывать данные
5. Нажать кнопку регистрация 

Ожидаемый результат: Сообщения это поле необходимо заполнить

Case 8 (Проверка на обязательное написание номера телефона при регистрации)
1. Перейти на сайт tickets.by/gd
2. Нажать на надпись Мой билет 
3. Нажать на кнопку зарегистрироваться 
4. Указать правильные данные , но не указывать номер телефона 
5. Нажать кнопку регистрация 

Ожидаемый результат: Сообщение об ошибке пожалуйста, введите корректный номер телефона


Case 9 (Проверка валидацию Email при регистрации )
1. Перейти на сайт tickets.by/gd
2. Нажать на надпись Мой билет 
3. Нажать на кнопку зарегистрироваться 
4. Указать правильные данные , но в поле Email указать не верный формат 
5. Нажать кнопку регистрация 


Ожидаемый результат: Предупреждение об ошибке пожалуйста, введите корректный адрес электронной почты


Case 10 (Проверка на изменение маршрута)
1. Перейти на сайт tickets.by/gd
2. Ввести место отправления – "Минск" 
3. Ввести место прибытия – "Борисов" 
4. Указать любую приемлемую дату 
5. Нажать кнопку "Поиск"
6. Нажать кнопку "Изменить условия поиска"
7. Ввести место прибытия – "Витебск"
8. Нажать кнопку "Поиск"

Ожидаемый результат: Новый выбор поездов
